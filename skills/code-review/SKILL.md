# Code Review

## Instructions
1. Read all the code in the repository.
2. Check the code against best practices for the programming language of the file.
3. Check the code is clear, easy to read and simple.
4. Check the code is consistent with the majority of the code in the project.
5. Suggest any refactoring opportunities.
